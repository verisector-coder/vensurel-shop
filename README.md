# 🛒 Mini App — веб-витрина Vensurel Shop

Это веб-версия магазина, которая открывается прямо внутри Telegram (сетка разделов,
карточки товаров, поиск). Покупка из витрины уходит в бота, и он оформляет заказ как обычно.

Файлы: `index.html` (витрина) + `catalog.json` (данные каталога).

## Как опубликовать (бесплатно, без навыков) — Netlify Drop
1. Открой https://app.netlify.com/drop
2. Перетащи туда **папку `miniapp`** целиком (с `index.html` и `catalog.json`).
3. Netlify выдаст ссылку вида `https://random-name.netlify.app` — это и есть адрес витрины.
4. В файле `.env` (папка funpay_bot) пропиши:
   `STORE_MINIAPP_URL=https://random-name.netlify.app`
5. Перезапусти бота. Внизу появится кнопка **«🛒 Открыть магазин»**, а в меню — `/miniapp`.

Альтернативы: GitHub Pages, Cloudflare Pages, Vercel — любой статический HTTPS-хостинг.
Требование Telegram: только HTTPS (Netlify/Pages дают его автоматически).

## Обновление каталога
Каталог в витрине берётся из `catalog.json`. После изменения товаров/остатков:
```
cd telegram_store
python export_catalog.py        # пересоберёт miniapp/catalog.json
```
Затем перезалей папку `miniapp` на хостинг (на Netlify — снова перетащи в Drop,
или подключи Git для авто-деплоя). Витрина подтягивает свежий `catalog.json` при каждом открытии.

## Как работает покупка
Кнопка «Купить» в витрине отправляет боту выбранный товар (Telegram `sendData`),
бот открывает оформление заказа: данные → оплата (баланс/CryptoBot) → автовыдача.
Покупка работает при запуске витрины через кнопку **«🛒 Открыть магазин»** (нижняя клавиатура).

## Настройка иконок (необязательно)
Хочешь свой логотип/маскота в шапке — замени эмодзи 🦝 в `index.html` (блок `.brand .dot`)
или вставь `<img>`.
